模除的运算

％ =

%号模除

当是正整数时:

源码:
#include "stdio.h"
int main(){
 printf("%d\n",7%3);//7÷3=2...1
 printf("%d\n",21%3);//21÷3=7...0
 printf("%d\n",21%10);//21÷10=2...1
 printf("%d\n\n",21%11);//21÷11=1...10
}				
				
运行结果:
1
0
1
10

模除的运算的结果:当被除数不够整除时余下的数.

当是负整数时:

源码:
#include "stdio.h"
int main(){
 printf("%d\n",-7%3);
 printf("%d\n",-21%3);
 printf("%d\n",-21%10);
 printf("%d\n\n",-21%11);
 
 printf("%d\n",7%-3);
 printf("%d\n",21%-3);
 printf("%d\n",21%-10);
 printf("%d\n\n",21%-11);
 
 printf("%d\n",-7%-3);
 printf("%d\n",-21%-3);
 printf("%d\n",-21%-10);
 printf("%d\n",-21%-11);
}
				
运行结果:
-1
0
-1
-10

1
0
1
10

-1
0
-1
-10

当%号前面是正数时,结果为正数

当%号前面是负数时,结果为负数

%号输出

源码:
#include "stdio.h"
int main(){  
    int x=2;   
    printf("%d\n",x);
    printf("%%d\n",x); 
    printf("$%d\n",x); 
    printf("*%d\n",x); 
    printf("%%%d\n",x); 
    printf("%%%%d\n",x);
    printf("%%%%%d\n",x); 
}  
				
运行结果:
2
%d
$2
*2
%2
%%d
%%2

任意符号+%,输出显示的都是任意符号.

%d就是取后面的值
