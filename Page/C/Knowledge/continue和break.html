C语言,||continue和break区别

更新时间:200623
C语言
continue
break
源码:
#include "stdio.h"
int main(){
    for(int a = 0; a < 7; a++){
        if(a%2==0){
            if(a==2){
                printf("continue1\n");
                continue;
                printf("continue2\n");
            }
            printf("能被2整除的数有:%d\n",a);
        }
        printf("(%d)",a);
    }
    printf("\n\n");
    for(int b = 0; b < 5; b++){
        if(b%2==0){
            if(b==2){
                printf("break1\n");
                break;
                printf("break2\n");
            }
            printf("能被2整除的数有:%d\n",b);
        }
        printf("(%d)",b);
    }
}
								
运行结果:
能被2整除的数有:0
(0)(1)continue1
(3)能被2整除的数有:4
(4)(5)能被2整除的数有:6
(6)

能被2整除的数有:0
(0)(1)break1
  • continue跳出当前语句
  • break跳出当前循环
