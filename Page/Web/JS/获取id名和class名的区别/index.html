获取id名和class名的区别

更新时间:200623
JS
id名
class名
css代码
div #box,.box{
	width: 100px;
	height: 100px;	
	margin: 10px auto;	
	color: #fff;	
	text-align: center;	
}
					
html代码
<div id="box" style="background: red;">box1</div>
<div class="box"  style="background: blue;">box2</div>

运行结果:

box1
box2
css代码:
div #box,.box{
	width: 100px;
	height: 100px;	
	margin: 10px auto;	
	color: #fff;	
	text-align: center;	
}									
					
html代码:
<div id="box box1" style="background: red; color: #fff;">box1</div>
<div class="box box2" style="background: blue;">box2</div>

运行结果:

box1
box2
  • 获取id名可以改变元素的样式
  • 获取class名不可以改变元素的样式,并且会报错
